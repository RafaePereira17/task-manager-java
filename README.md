# 🗂️ Gerenciador de Tarefas em Java

Projeto simples de CRUD usando Java + SQLite.

## 🚀 Funcionalidades

- Adicionar tarefa
- Listar tarefas
- Marcar tarefa como concluída
- Remover tarefa

## 🛠️ Tecnologias

- Java
- SQLite
- JDBC
